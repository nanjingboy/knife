# knife
Some widgets I often used in projects for Android

# Setup

* In your `build.gradle` :

```gradle
repositories {
    maven { url "https://jitpack.io" }
}

dependencies {
    implementation 'com.github.nanjingboy:knife:1.0.7'
}
```
